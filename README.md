# Bookmark Cli


## Getting Started


### Prerequisites

```
python 3
```

### Installing

```
$ ./run.py -t test
please input your urls, if you are done just type "done"
[*] urls: https://google.com
[*] urls: https://github.com
[*] urls: done
```
#### 
```
$ ./run.py -v
test
```
